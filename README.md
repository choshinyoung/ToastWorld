# Cellular Automata DSL Sandbox

## 🌌 서문: 게임을 위한 게임 (Game for Game's Sake)
과거의 예술이 생존을 위한 수단에서 벗어나 '예술 그 자체'를 위한 행위로 승화되었듯, 이 프로젝트는 클리어해야 할 목표나 점수 따위의 세속적 목적을 거부합니다.

이곳에는 이겨야 할 적도, 엔딩도 존재하지 않습니다. 오직 당신이 창조한 **규칙(Code)**과 그로 인해 태어나는 **현상(Phenomenon)**만이 존재할 뿐입니다. 이 샌드박스는 플레이어가 직접 세계의 물리 법칙을 코딩하고, 그 안에서 피어나는 무한한 패턴을 감상하는 **순수 유희이자 디지털 캔버스**입니다.

## 💎 이 프로그램의 위대함

### 1. 창조의 언어, 독자적 DSL (Domain Specific Language)
[cite_start]기존 게임이 개발자가 정해둔 규칙에 종속되는 경험이라면, 이 프로그램은 **자체 개발된 파서(Parser)**를 통해 플레이어에게 신의 권능을 부여합니다[cite: 17].
* [cite_start]`sand is fallable`, `life is { color = "#4ade80" }`와 같은 직관적인 문법으로 존재의 본질을 정의합니다[cite: 7, 13].
* [cite_start]단순한 블록 배치를 넘어, 입자의 물성과 상호작용 메커니즘을 근본부터 설계하는 예술적 자유를 보장합니다[cite: 15, 20].

### 2. 살아 숨 쉬는 캔버스 (Live Coding Environment)
정지된 그림이 아닌, 매 순간 변화하는 역동적인 예술을 지향합니다.
* [cite_start]시뮬레이션이 돌아가는 와중에도 실시간으로 코드를 수정하고(`editCode`), 그 결과가 즉각적으로 세계에 반영됩니다[cite: 106, 112].
* [cite_start]전역적인 법칙(`Global Types`)과 개별 세포의 돌연변이(`Custom Definition`)가 공존하며, 예측 불가능한 카오스의 아름다움을 만들어냅니다[cite: 70, 81].

### 3. 복잡계의 미학 (Emergent Complexity)
단순한 세포 하나하나가 모여 거대한 흐름을 만들어내는 '창발성(Emergence)'을 구현했습니다.
* [cite_start]**Conway's Game of Life**의 로직을 포함하여, 주변(`nearby`)을 인식하고, 접촉(`touching`)하며, 스스로 상태를 변화(`become`)시키는 고도의 상호작용 엔진을 탑재했습니다[cite: 48, 53, 57].
* [cite_start]단순한 이동(`move`) 함수가 모여 중력, 유체 역학, 생명 활동과 같은 거시적인 물리 현상을 예술적으로 시각화합니다[cite: 58].
